# Product glossary

Terms I use consistently across products

- **command palette**: UX to interact with any part of the product. Generally via a textbox that can be opened anytime with a keyboard shortcut and/or UI button. Suggests results based on fuzzy matches to the user's input. If no input, suggests general results or recent commands.
- **log in**: See "sign in"
- **log out**: See "sign out"
- **open-source**: The source code of the product is freely available for others to copy. Similar to FOSS. Preferred over "open source."
- **settings**: All configurations specific to that person's use of the product. Preferred over "configuration."
- **sign in**: Enter credentials to view personalized information. Preferred over "log in."
- **sign out**: Remove credentials to hide personalized information. Preferred over "log out."
