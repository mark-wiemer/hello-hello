# Hello Bun

[Bun](https://bun.sh/) is an all-in-one JavaScript, TypeScript, and TSX toolkit. It includes:

- a runtime
- a test framework
- a package manager
- a bundler
- probably more!

With Node, developers have to also install `pnpm` to manage packages, `vitest` to test large projects (though Node now has its own test framework), and `vite` to bundle scripts. While [Vite+](https://viteplus.dev/) is aiming to do this, it's in allow-listed early access as of writing, and doesn't look like it'll be fully open-source.

I use Bun because CodeCrafters uses Bun.
