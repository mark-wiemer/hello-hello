# Hello style

Style guide for writing 🤓 This covers what, when, and how, but not why.

## Citations

Use "datetime, title, author, platform" structure if all values are important:

[2026-01-27, 07:58 PST - Crying over breakfast - Mark Wiemer on markwiemer.com](https://markwiemer.com/blog/crying-over-breakfast/)

If some values (except title) are redundant or not necessary, we can remove them:

[Crying over breakfast - markwiemer.com](https://markwiemer.com/blog/crying-over-breakfast/)

If the indvidual author is employed by their platform, like a journalist, use `__author__ for __platform__`. If the author is not relevant and the platform is a news source, we can simply use the platform name.

If there are multiple authors for a specific platform (a news source but not a social media site), we can simply state the platform.

## Referring to the audience

Use "we" to refer to anyone reading the document, unless a separation needs to be made between the author and the audience:

- > To learn how to cook, we can open YouTube and search "cooking tips."

- > Maintainers: we can go to our admin portal to manage the website.

- > I love dogs, but it's totally fine if you don't!

## Spelling

Prefer the most common spelling for the audience. If it's unclear, follow these guidelines:

- built-in, not builtin
- open-source, not open source
- semicolon, not semi-colon
