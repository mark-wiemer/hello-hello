# Luanti

[Luanti](https://docs.luanti.org) (formerly Minetest) is an open-source ["boxel"](/glossary#boxel) game creation system. It has three parts: an engine for making and modding games similar to Minecraft, a distribution platform for creators, and an application for players. As an open-source project, Luanti games and mods will live for as long as someone has them installed and is willing to share. Usage of Luanti is completely free, and anyone can make copies or modifications to their heart's content, as long as they follow [Luanti's free and open-source license](https://github.com/luanti-org/luanti/blob/master/LICENSE.txt).

Luanti was [started in 2010](https://github.com/luanti-org/luanti/commits/?since=2010-11-14&until=2010-11-27) and active development continues today! I became a contributor in December 2024 when Luanti contributors started [docs.luanti.org](https://docs.luanti.org). Through early 2025, [I helped migrate content](https://github.com/luanti-org/docs.luanti.org/commits/master/?author=mark-wiemer&since=2024-12-01&until=2025-03-31), fix typos, and categorize pages.

In late 2025, I joined the initiative to improve Luanti's API docs. My goal is to [rewrite the docs](https://github.com/luanti-org/docs.luanti.org/issues/296) into a domain-specific language so that creators can use IntelliSense and developers can translate the API into other programming languages. This is a huge ongoing effort but we are making great progress!

Anyone is welcome to contribute to Luanti! The [doc source](https://github.com/luanti-org/docs.luanti.org) is hosted on GitHub and build instructions are in the readme :) The [engine source](https://github.com/luanti-org/luanti) is not my forte but you're welcome to try and ask me questions!

## Subprojects

Projects are listed alphabetically.

### doc-schema

Make the docs parsable for extensions, plugins, and content type transformations.
[doc-schema on GitHub](https://github.com/mark-wiemer/hello-hello/tree/main/packages/luanti/doc-schema)

### docs.luanti.org

[Official doc site](https://docs.luanti.org) launched December 2024.
[docs.luanti.org on GitHub](https://github.com/luanti-org/docs.luanti.org)

### luanti

Source code for the game engine and application for players.
[luanti on GitHub](https://github.com/luanti-org/luanti)

### luanti-ts-api

Handwritten TypeScript API for Luanti.
[luanti-ts-api on GitHub](https://github.com/mark-wiemer/hello-hello/tree/main/packages/luanti/luanti-ts-api)

### sample-mod

A minimal hello world mod to learn about the current tooling for mods written in Lua.
[Enable the LuaLS addon](https://luals.github.io/wiki/addons/#addon-manager) for "Luanti" and "Luanti Full API".
Also consider using Luanti Tools.
You shouldn't see any yellow or red squiggles.
[sample-mod on GitHub](https://github.com/mark-wiemer/hello-hello/tree/main/packages/luanti/sample-mod)

### void-game-ts

A very minimal game written in TypeScript alongside my handwritten TypeScript API.
[void-game-ts on GitHub](https://github.com/mark-wiemer/hello-hello/tree/main/packages/luanti/void-game-ts)
