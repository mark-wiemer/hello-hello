# GitHub

Many services beyond just hosting repos. I work for Microsoft, which owns GitHub, so I'm biased. Happy to consider Codeberg, Gitea, GitLab, or others if they have reasonable feature sets!

- GitHub Actions automates workflows
  - To best automatically add items to a project, see [Auto-add issues to project (hello-hello#40) - GitHub](https://github.com/mark-wiemer/hello-hello/issues/40)
- [GitHub Apps](/software/github/apps)
- GitHub Projects helps manage issues and PRs
  - Automation with user-owned projects is limited, consider org-owned projects instead. See [Question about personal access token (actions/add-to-project#306) - GitHub](https://github.com/actions/add-to-project/issues/306)

See also [Software - markwiemer.com](/software)
