# Mocha 12

Mocha 12 (sometimes "v12") is an upcoming release of [Mocha](/software/mocha) which updates internal dependencies while maintaining a stable end-user experience. There were originally no planned breaking changes between Mocha 11 and Mocha 12, but we added a few to spice things up (and make everyone's life easier in the long run).

We released 11 unique betas through early April 2026, and I was personally aiming for a release candidate in early April and a full release in late April. However, I got super sick in March and didn't recover until late May. It's now mid-June and we plan to have the first release candidate ready this weekend (by June 20). We haven't yet decided how long to wait between the first release candidate and the stable release.

Dates and times are PDT (UTC-7). All links are to GitHub unless otherwise specified.
