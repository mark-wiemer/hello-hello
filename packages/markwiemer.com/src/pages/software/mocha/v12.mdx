# Mocha 12

Mocha 12 (sometimes "v12") is an upcoming release of [Mocha](/software/mocha) which updates internal dependencies while maintaining a stable end-user experience. There were originally no planned breaking changes between Mocha 11 and Mocha 12, but we added a few to spice things up (and make everyone's life easier in the long run).

We released 11 unique betas through early April 2026, and I was personally aiming for a release candidate in early April and a full release in late April. However, I got super sick in March and didn't recover until late May. It's now mid-June and we plan to have the first release candidate ready this weekend (by June 20).

At this point, there is so little remaining work that we can track it all within a single GitHub issue!
[#5357: Mocha 12 release plan](https://github.com/mochajs/mocha/issues/5357)
