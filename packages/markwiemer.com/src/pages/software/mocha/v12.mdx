# Mocha 12

Mocha 12 (sometimes "v12") is an upcoming release of [Mocha](/software/mocha) which updates internal dependencies while maintaining a stable end-user experience. There are no planned breaking changes between Mocha 11 and Mocha 12.

We released 11 unique betas through early April 2026, and I was personally aiming for a release candidate in early April and a full release in late April. However, I got super sick.

Now that I'm healthy, the plan is simple:

1. Slowly migrate from CJS to ESM
1. Review unrelated pending PRs while I wait
1. Once we're converted to ESM, release the first Mocha 12 release candidate (12.0.0-rc.1)
1. Iteratively fix bugs, but no new features in any release candidates
1. Once we're confident in stability, release Mocha 12!

Dates and times are PDT (UTC-7). All links are to GitHub unless otherwise specified.
