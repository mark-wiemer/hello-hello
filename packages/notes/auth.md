## Glossary

- [application layer](https://en.wikipedia.org/wiki/Application_layer): specifies the communications protocols (application protocols) of a communications network
- [communications system](https://en.wikipedia.org/wiki/Communications_system): everything needed to send a signal from sender to receiver such that the receiver can understand it
- [DAP](https://en.wikipedia.org/wiki/Directory_Access_Protocol): Directory Access Protocol
- [directory service](https://en.wikipedia.org/wiki/Directory_service): maps names of resources to network addresses
- [Internet protocol suite](https://en.wikipedia.org/wiki/Internet_protocol_suite): aka TCP/IP, also includes UDP
- [IP](https://en.wikipedia.org/wiki/Internet_Protocol): Internet Protocol
- [ISO](https://en.wikipedia.org/wiki/International_Organization_for_Standardization): International Organization for Standardization (intentionally abbreviated such that its name is not an acronym in any language)
- [LDAP](https://en.wikipedia.org/wiki/Lightweight_Directory_Access_Protocol): lightweight directory access protocol, open-source application protocol for directory services, usually storing usernames and passwords. Subset of X.500 standards including X.511 (DAP)
- [Kerberos](<https://en.wikipedia.org/wiki/Kerberos_(protocol)>): open-source authentication protocol
- [network address](https://en.wikipedia.org/wiki/Network_address): ID for a node in a network (e.g. IP address)
- [NTLM](https://en.wikipedia.org/wiki/NTLM): New Technology LAN Manager, a closed-source suite Microsoft security protocols
- [OSI model](https://en.wikipedia.org/wiki/OSI_model): Open Systems Interconnection model by ISO, contains 7 abstraction layers describing a communication system: Physical, Data Link, Network, Transport, Session, Presentation, and Application
- [TCP](https://en.wikipedia.org/wiki/Transmission_Control_Protocol): Transmission Control Protocol
- [UDP](https://en.wikipedia.org/wiki/User_Datagram_Protocol): User Datagram Protocol
- [X.500](https://en.wikipedia.org/wiki/X.500): networking standards for directory services
