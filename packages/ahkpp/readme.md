# AHK++

I maintain [AHK++](https://ahkpp.com), an an AHK language support extension for AHK v1 and v2, supporting IDEs like [VS Code](https://code.visualstudio.com/), [VSCodium](https://vscodium.com/), and [Theia IDE](https://theia-ide.org/).
